## Usage

1. Go to `https://yourinstance.tld/settings/admin/instance/settings`
2. Scroll down to the bottom
3. Paste the content of [rose-pine-main-dawn.css](https://github.com/rose-pine/gotosocial/blob/main/dist/rose-pine-main-dawn.css) or [rose-pine-moon-dawn.css](https://github.com/rose-pine/gotosocial/blob/main/dist/rose-pine-moon-dawn.css) into Custom CSS
4. Click `Save`
